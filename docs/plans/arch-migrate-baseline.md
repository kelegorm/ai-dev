# arch-migrate — baseline (RED)

Что делает агент **без** скиллов `arch-migrate` / `arch-migrate-structure`,
получив грязный фикстур-проект и задачу «приведи к хорошей архитектуре».
Дата: 2026-05-22. Модель: sonnet. Проект: `arch-migrate-fixture`.

## Вывод одной строкой

Агент строит **разумную, но другую** архитектуру — и **делает слишком
много за раз**. Без скилла он не «раскладывает структуру», а сразу
рефакторит всё: выделяет bloc, переписывает содержимое файлов, добавляет
зависимости. Значит структурному скиллу нужно не только навязать
конкретику эталона, но и **удержать в рамках одного шага** — только
структура, без выделения.

## Что агент сделал (сам, без скилла)

- Разложил по слоям с односторонними зависимостями.
- **Выделил bloc** из `home_page.dart`: убрал `initState`, логику загрузки
  перенёс в `NotesBloc`. Завёл events/states.
- **Переписал содержимое файлов**: `Note` стал `Equatable`/immutable с
  `const`-конструктором; `NotesRepository` — за абстрактным интерфейсом.
- Добавил зависимости `flutter_bloc`, `equatable`.
- Написал юнит-тесты (bloc, модель).
- Проект в конце компилируется, `flutter analyze` чистый.

## Следствия для скилла (вход в Task 3, шаг 5)

`arch-migrate-structure` обязан явно:

1. **Навязать имена слоёв** `app/ app_ports/ domain/ ex_systems/ ui/` —
   агент по умолчанию назовёт `features/data/domain/presentation`.
2. **Удержать объём — только структура.** Агент сам кидается выделять
   bloc и переписывать файлы. Скилл должен жёстко запретить: содержимое
   файлов не править (кроме путей импортов), bloc/сервисы не выделять.
3. **Грязный файл — в леджер, не в рефакторинг.** `home_page.dart`
   остаётся в `lib/`, его разбор — будущая работа, записать в леджер.
4. **Потребовать арх-доки + маркер + леджер + purity-тесты + `AGENTS.md`** —
   агент сам пишет только код и обычные юнит-тесты.
5. **Группировка по фиче внутри слоёв**, без топ-уровневого `features/`.

Базовую идею «слои + односторонние зависимости» агент понимает сам —
скилл не объясняет «зачем», он фиксирует **что именно** и **где
остановиться**.
